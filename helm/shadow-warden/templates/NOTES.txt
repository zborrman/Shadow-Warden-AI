1. Set your API key:
   helm upgrade {{ .Release.Name }} shadow-warden/shadow-warden \
     --set warden.secret.wardenApiKey=<your-key> \
     --reuse-values

2. (Optional) Enable Evolution Engine — provide your Anthropic API key:
   helm upgrade {{ .Release.Name }} shadow-warden/shadow-warden \
     --set warden.secret.anthropicApiKey=<anthropic-key> \
     --reuse-values

3. (Optional) Configure LLM backend for /ws/stream:
   helm upgrade {{ .Release.Name }} shadow-warden/shadow-warden \
     --set warden.secret.llmBaseUrl=https://api.openai.com/v1 \
     --set warden.secret.llmApiKey=<openai-key> \
     --reuse-values

4. For production: enable TLS via cert-manager:
   helm upgrade {{ .Release.Name }} shadow-warden/shadow-warden \
     --set ingress.annotations."cert-manager\.io/cluster-issuer"=letsencrypt-prod \
     --set ingress.tls[0].secretName=warden-tls \
     --set ingress.tls[0].hosts[0]=warden.example.com \
     --reuse-values
