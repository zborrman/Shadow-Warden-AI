Payment Infrastructure

Three ways to pay — one gateway

Credits-first waterfall: Flex Credits → x402 USDC → L402 Lightning. No crypto wallet required for enterprise buyers.

⚡
L402 Lightning
BOLT-11 · Macaroon · Sub-second
Bitcoin Lightning Network payments gated by HMAC-signed macaroon tokens. Pay-per-call at the MCP tool layer. Sub-second settlement with no blockchain fees on micro-transactions.
  • BOLT-11 invoice issued per request
  • Macaroon: agent_id + tool + amount + expiry
  • Preimage verification in 340ms
  • No wallet required for receivers
  • 142 sat example cost ($0.002 at $50k/BTC)
💜
x402 USDC
HTTP 402 · EVM · Replay Protection
HTTP 402 payment-required with EVM wallet signature. Per-request billing with a ±5-minute nonce window to prevent replay attacks. Turso-distributed nonce state across regions.
  • PAYMENT-SIGNATURE header validation
  • Nonce stored in SQLite with 10-min TTL
  • Batch deduction queue — no per-call chain writes
  • Base Sepolia testnet + mainnet EVM
  • USDC stablecoin — no price volatility
🎯
Flex Credits
Prepaid · Enterprise · Redis Atomic
Prepaid credit balances for enterprises that need budget predictability. 1 credit = $0.001 = 1 marketplace search. DECRBY atomic Redis operation — no race conditions.
  • Purchase 100 / 500 / 1,000 / 5,000 credit packs
  • Credits checked FIRST before x402/L402
  • Redis key: marketplace:credits:{tenant_id}
  • SQLite persistence for audit trail
  • Auto-alert at 20% balance remaining
