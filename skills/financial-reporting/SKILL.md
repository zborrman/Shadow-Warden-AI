# Skill: Financial Reporting — ClearingEngine Statements

**Purpose:** Generate detailed financial statements from `marketplace_clearing_log` for B2B clients. Covers GMV, platform take rate (1.5%), seller net proceeds, and First-Proposal Bias efficiency. Output: structured JSON, Markdown, or PDF.

---

## When to invoke

- A tenant requests a period billing summary via `POST /marketplace/analytics/query`
- SOVA `sova_sla_report` (Monday 09:00 UTC) needs a financial section
- Monthly MasterAgent `ComplianceAgent` audit run
- A B2B client requests an invoice-grade clearing statement

---

## Data sources

| Table | Key fields | Notes |
|-------|-----------|-------|
| `marketplace_clearing_log` | `platform_fee_usd`, `seller_net_usd`, `cleared_at` | Primary billing source |
| `marketplace_negotiations` | `agreed_price`, `buyer_agent_id`, `seller_agent_id` | Trade detail |
| `marketplace_purchases` | `price_paid`, `buyer_agent`, `seller_agent`, `asset_type` | Revenue by category |
| `marketplace_agents` | `name`, `tenant_id`, `did` | Agent identity |

All queries go through `POST /marketplace/analytics/query` with `caller_agent_id` scoped to the requesting tenant.

### SQL queries (safe — all values parameterized)

**GMV and take rate summary:**
```sql
SELECT
  SUM(seller_net_usd + platform_fee_usd) AS gmv_usd,
  SUM(platform_fee_usd)                   AS total_fee_usd,
  SUM(seller_net_usd)                     AS total_seller_net,
  COUNT(*)                                AS trades,
  AVG(seller_net_usd + platform_fee_usd)  AS avg_trade_usd
FROM marketplace_clearing_log
WHERE cleared_at >= ?
  AND cleared_at < ?
```

**First-Proposal Bias metric:**
```sql
SELECT
  COUNT(CASE WHEN round_count = 1 THEN 1 END) * 1.0 / COUNT(*) AS acceptance_rate,
  AVG(round_count)                                               AS avg_rounds
FROM marketplace_negotiations
WHERE status = 'accepted'
  AND created_at >= ?
  AND created_at < ?
```

## Decimal precision

All monetary aggregation from the DB returns `REAL` (float). Re-quantize before displaying:
```python
from decimal import Decimal, ROUND_HALF_UP
gmv = Decimal(str(row["gmv_usd"])).quantize(Decimal("0.01"), rounding=ROUND_HALF_UP)
```

Never display raw `float` values in financial statements.
