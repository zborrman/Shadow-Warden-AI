Data Handling Principles

Zero Content Logging

Data stays in your jurisdiction. The sovereign routing engine enforces 8-jurisdiction transfer rules, adequacy decisions, and PHI restrictions.

GDPR Art. 17 erasure runs as an ARQ cron. Request-level purge available via API. Data minimisation enforced pipeline-wide.

Data Residency Guarantee

All sub-processors that handle EU personal data are covered by Standard Contractual Clauses (SCCs) or operate within adequacy-decision jurisdictions. The Sovereign Routing Engine ensures PHI and CLASSIFIED data never leaves the configured jurisdiction. Classified data transfer is hardcoded to BLOCKED in warden/sovereign/jurisdictions.py.
