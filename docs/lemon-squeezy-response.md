- **Delivery:** Cloud service, API access + web dashboard. No physical goods.
- **Billing model:** Monthly or annual subscriptions (recurring). Add-ons are separate recurring charges.
- **Customers:** B2B — SMBs, development teams, and enterprises that use LLMs (OpenAI, Anthropic, etc.) and need security/compliance controls.
- **Trial:** 14-day free trial on all paid plans, no card required.
