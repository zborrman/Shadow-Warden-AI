Detection Engine
Configure the semantic filter pipeline thresholds and behaviour
ML Brain (MiniLM)
Semantic Threshold
Cosine similarity score above which a request is flagged by the ML model.
Lower = more sensitive (more false positives). Default: 0.72
0.72
Strict Mode
In strict mode, MEDIUM risk requests are also blocked (not just HIGH/BLOCK). Increases security at the cost of higher false-positive rate.
Compound Risk Escalation
3 or more MEDIUM signals in a single request auto-escalate to HIGH. Always enabled — prevents compound jailbreaks.
Always On

API Keys
Per-tenant key management with SHA-256 hash lookup
Authentication
Auth Mode
When WARDEN_API_KEYS_PATH is set: JSON multi-key file with per-tenant rate limits. When WARDEN_API_KEY is set: single shared key mode.
Multi-Key
Key Format
Keys are stored as SHA-256 hashes — plaintext is never persisted. Comparison is constant-time to prevent timing attacks.
SHA-256 Hashed
Key File Format
Set WARDEN_API_KEYS_PATH to a JSON file:
{
  "keys": [
    {
      "key_hash": "sha256-of-key",
      "tenant_id": "acme-corp",
      "rate_limit": 120
    }
  ]
}
NVIDIA Integration
NIM model routing and Agent Toolkit policy runtime for autonomous agents
Credentials
NVIDIA API Key
Generate at build.nvidia.com → API Keys. Required for NIM routing.
⚠ Restart required after saving
NIM Endpoint URL
All nim/-prefixed model requests are forwarded here.
NIM Model Routing
Enable NIM Routing
Route nim/<org>/<model> requests through the Shadow Warden filter pipeline before forwarding to NVIDIA.
Default NIM Model
Used when no model is specified in agentic calls. Any NVIDIA API catalog model ID.
NIM Embeddings
Route POST /v1/embeddings with nim/ prefix to NVIDIA embeddings endpoint.
NVIDIA Agent Toolkit
Enable Agent Toolkit Integration
Activates policy-based safety rails for autonomous agent workflows. Wraps every agentic call with input/output guardrails.
Prompt Shield
Detects indirect prompt injection, tool-call hijacking, and persona switches in agentic workflows. Requires PROMPT_SHIELD_ENABLED=true.
Cryptographic Audit Trail
SHA-256 hash chain for every agentic decision — tamper-evident, SOC 2 compliant. Requires AUDIT_TRAIL_ENABLED=true.
Block on Agent Toolkit Policy Violation
Hard-block requests that violate Agent Toolkit input/output rails. When off, violations are logged but forwarded.
Quick-Start
1 — Install
pip install nemo-guardrails
2 — config/config.yml
models:
  - type: main
    engine: openai
    model: nim/nvidia/llama-3.1-nemotron-70b-instruct
    parameters:
      base_url: http://localhost:8001/v1
      api_key: ${WARDEN_API_KEY}
3 — Python
from nemoguardrails import RailsConfig, LLMRails
rails = LLMRails(RailsConfig.from_path("./config"))
result = await rails.generate_async(messages=[...])
4 — .env
NVIDIA_API_KEY=nvapi-…
PROMPT_SHIELD_ENABLED=true
AUDIT_TRAIL_ENABLED=true
API key and endpoint changes require a service restart.
