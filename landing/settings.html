ML Uncertainty Escalation

ML scores in [lower, semantic_threshold) are flagged as ML_UNCERTAIN and escalated to MEDIUM risk. Set to 0 to disable. Default: 0.55.

from nemoguardrails import RailsConfig, LLMRails
from warden.integrations.nemo_bridge import create_nemo_integration, DualOutputGuard

bridge, synthesizer = create_nemo_integration()
rails = LLMRails(RailsConfig.from_path("./warden/integrations/nemo_colang"))
synthesizer.register_rails(rails)

async def handle(user_msg):
    result = await warden.filter_async(user_msg)
    if not result.allowed: return "Blocked."
    ctx = bridge.to_colang_context(result) # geometry → Colang vars
    resp = await rails.generate_async(messages=[{"role":"user","content":user_msg}], context=ctx)
    return await DualOutputGuard(bridge).check_output(user_msg, resp, warden)

Betti Number Weights
Noise Score Formula
Final noise score = 0.33×char + 0.27×word + 0.22×diversity + 0.10×β₀ + 0.08×β₁. β₀ = connected components (0-cycles), β₁ = 1-cycles. Weights are fixed.
Fixed Weights
Ripser (True Persistent Homology)
When ripser is installed, exact Betti numbers are computed. Otherwise algebraic approximations are used (< 0.1% accuracy difference, 3× faster).
—

Blend Weight
Hyperbolic Weight
