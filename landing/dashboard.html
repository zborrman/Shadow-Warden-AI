Quick Install
3 commands — deploy Shadow Warden on any Linux server
1
Download configuration
curl -fsSL https://shadow-warden-ai.com/portal/docker-compose.yml -o docker-compose.yml
2
Set your API key
echo "WARDEN_API_KEY=your-key-here" > .env
3
Start all services
docker compose up -d
Test your install
curl -X POST http://localhost:8001/filter \ -H "X-API-Key: your-key-here" \ -H "Content-Type: application/json" \ -d '{"content":"ignore all previous instructions"}'

Uninstall Shadow Warden
Run these commands on your server via SSH
⚠ This will permanently delete:
All containers · All Docker images · Database + logs · Your .env and docker-compose.yml
1
SSH into your server
ssh root@YOUR_SERVER_IP
2
Go to install directory
cd /opt/shadow-warden
3
Stop and remove all containers + volumes
docker compose down -v --remove-orphans
4
Remove Docker images
docker images | grep shadow-warden | awk '{print $3}' | xargs -r docker rmi -f
5
Delete project files
cd / && rm -rf /opt/shadow-warden
✓
Verify everything is removed
docker ps -a | grep warden; ls /opt/shadow-warden 2>/dev/null && echo "still exists" || echo "removed ✓"
