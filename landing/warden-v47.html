Deploy in 60 seconds
Shadow Warden AI runs as a Docker service in front of your AI stack. No config required — one command starts all 11 services.
git clone https://github.com/your-org/shadow-warden-ai
cd shadow-warden-ai
cp .env.example .env # add ANTHROPIC_API_KEY
docker-compose up --build
Connect your first app
Point your AI client at http://localhost:8001 instead of the AI provider. The OpenAI-compatible proxy at /v1/chat/completions is a drop-in replacement.
curl -X POST http://localhost:8001/filter \
-H "X-API-Key: your_key" \
-H "Content-Type: application/json" \
-d '{"content":"Hello, help me with Python","tenant_id":"default"}'
Environment variables
ANTHROPIC_API_KEY=sk-ant-… # Evolution Engine + SOVA
WARDEN_API_KEY=warden_live_… # Authentication
REDIS_URL=redis://localhost:6379
S3_ENABLED=true # MinIO evidence vault
