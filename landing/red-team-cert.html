FAQ

Common questions

How much time commitment per week?

Expect 6–8 hours per week: ~2h theory, ~3h lab, ~1h live Q&A. The capstone week is more intensive (~15h for the 72h exercise). All content is async except Thursday live sessions.

Do I need GPU or special hardware?

No. All compute-heavy labs run in our cloud lab environment (pre-configured Docker containers). You only need a browser and Python installed. Shadow Warden itself runs CPU-only.

Is this ethical / legal?

100% yes. All labs target systems we own and operate exclusively for this purpose. The program teaches offensive techniques exclusively in authorized, isolated environments — the same way CEH, OSCP, and PNPT do. No real systems are ever targeted without explicit authorization.

Will this count toward OSCP / CEH CPE credits?

We're working toward formal accreditation. The Shadow Warden certificate won't count for CPE automatically in the pilot phase, but we'll provide a detailed completion letter with learning objectives for manual CPE submission.

What if I miss a live session?

All live Q&A sessions are recorded and posted to the cohort portal within 2 hours. You can submit questions in advance via Slack. Missing up to 3 sessions won't affect certification eligibility.
